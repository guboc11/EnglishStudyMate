# Strategy

## Goal
8주 내 영어 학습 MVP를 출시하고, 2~4주 사용한 초기 사용자가 학습한 단어·표현이 포함된 영상/콘텐츠를 막힘없이 이해하고 자신의 언어로 설명할 수 있는 수준(자연 회상/자연 사용)에 도달하게 한다.

## Focus
- 학습한 단어·표현을 실제 맥락에서 자연스럽게 이해하고 설명하는 경험에 집중한다.
- 단순 정답 맞추기보다, 같은 표현을 다양한 상황에서 다시 만나며 의미 연결이 강화되는 학습 흐름에 집중한다.
- 짧은 기간(2~4주) 안에도 사용자가 스스로 "이제 이 표현은 익숙하다"고 느낄 수 있는 체감 성과를 만드는 데 집중한다.

Hypothesis: 이미 여러 번 경험한 표현을 재조합한 콘텐츠를 제공하면, 자연 회상과 이해도가 높아진다.

## Non-focus
- 초기 단계에서 문법 전 범위 완성이나 시험 대비 중심 기능은 우선순위에서 제외한다.
- 기능의 양을 빠르게 늘리는 것보다 핵심 학습 경험의 완성도를 우선한다.
- 다언어 확장은 장기 방향으로 유지하되, 현재 단계에서는 영어 학습 경험 검증에 집중한다.

## Success Metrics
- 첫째, 학습한 단어·표현이 포함된 영상 또는 짧은 콘텐츠를 사용자가 실제로 얼마나 정확하게 이해하는지를 본다. 단순 객관식 정답률이 아니라, 사용자가 해당 장면의 의미를 자신의 말로 설명했을 때 핵심 의미를 놓치지 않는지를 기준으로 삼는다.
- 둘째, 사용자가 학습한 표현을 "기억해내는" 수준을 넘어 "자연스럽게 떠올리고 쓰는" 수준에 도달했는지를 본다. 이를 위해 같은 표현을 간격을 두고 재노출했을 때 별도 힌트 없이도 이해·설명이 가능한 비율을 추적한다.
- 셋째, 2~4주 학습 기간 동안 사용자가 학습 흐름을 실제로 유지했는지를 본다. 학습 지속률과 세션 완료율을 함께 확인해, 학습 효과가 특정 소수 사용자에만 나타나는지 아니면 일반적인 초기 사용자군에서도 재현되는지를 판단한다.
- 넷째, 사용자가 체감하는 학습 변화가 분명한지를 정성적으로 확인한다. 예를 들어 "예전에는 멈췄던 표현이 이제는 바로 이해된다"와 같은 자기보고를 수집해 정량 지표와 함께 해석한다.
