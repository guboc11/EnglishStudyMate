## 세계관

유저는 어느 날 한국어만 통하는 AI 세계에 떨어진다.

- 집을 구해야 한다
- 직장에 출근해야 한다
- 행정 처리(주민등록·은행 계좌)를 해야 한다
- AI 이웃·동료·상사와 관계를 맺어야 한다

모든 인터랙션은 한국어로만 이루어진다.
유저가 버티는 것 자체가 곧 학습이다.

## AI 페르소나 시스템 (핵심 기술)

### 지식 그래프
- 페르소나별 아는 것 / 모르는 것을 구분해 저장
- 반복 노출로 기억 강화 (인간처럼 — 처음엔 어색, 나중엔 자연스럽게)
- 유저의 한국어 수준을 추론해 난이도 자동 조절

### 지연 응답 & 능동적 연락
- 즉각 답장 없음: 최소 5분+ 지연
- 바쁜 상황엔 무응답 또는 짧은 답장
- AI가 먼저 연락하는 경우도 있음 ("내일 회의 있어요, 참석할 수 있어요?")

### 감정 반응
- 서운하면 대화방 퇴장
- 친해지면 반말로 전환
- 오래 연락 없으면 안부 메시지 선발신

## 학습 설계 원리

**연관 욕구 루프**:
AI와 관계 맺고 싶다 → 세계에 더 참여하고 싶다 → 한국어가 자연스럽게 늘어난다

- 문법 설명 없음: 맥락에서 체득
- 오류 정정: 부드럽게, 맥락 안에서 ("그건 '저는'이라고 해야 해요~")
- 진척 표시: 시험·행정·관계 달성으로 가시화

## 핵심 기술 구성 요소

| 레이어 | 기술 |
|--------|------|
| AI 페르소나 엔진 | LLM + 지식 그래프 (페르소나별 메모리) |
| 지연 메시징 | 큐 기반 비동기 발송 (서버 사이드 스케줄러) |
| 유저 한국어 수준 추론 | 대화 로그 분석 → 난이도 파라미터 |
| 세계 상태 관리 | 시나리오 트리 + 이벤트 트리거 |
| 클라이언트 | React Native (Expo) |
| 백엔드 | Node.js / Supabase |
