## Scene 3 — 숙제 제출 (📝)

Alex가 숙제 탭을 탭한다.

### 숙제 A — 느낀점 일기

```
📝 오늘의 숙제

오늘 춘향전에서 인상 깊었던 장면을
1~2문장으로 써보세요.

┌────────────────────────┐
│                        │
│                        │
└────────────────────────┘

                    [제출하기]
```

Alex가 입력한다:

```
춘향이 "죽어도 싫소"라고 말한 장면이
인상 깊었어요. 용감한 사람이에요.
```

[제출하기] 버튼을 누른다.

```
토스트: 선생님이 확인 중입니다 🌟
```

---

### 숙제 B — 손글씨 사진 (같은 탭에 이어서)

```

하나를 종이에 손으로 쓰고 사진으로 제출하세요.

  [📷 사진 찍기]

한국어 단어:
┌────────────────────────┐
│                        │
└────────────────────────┘

                    [제출하기]
```

Alex가 📷 버튼을 탭한다.

```
시뮬레이션: 카메라 권한 요청 중...
[미리보기: 손글씨 "반드시" placeholder]

```


**오늘의 학교 완료!** 방과 후 활동 잠금 해제.
