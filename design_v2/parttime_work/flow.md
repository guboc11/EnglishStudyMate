# 파트타임 워크 v2 — Flow

## 레이아웃 구조

```
┌──────────┐
│          │
│          │
│  핸드폰  │
│  프레임  │
│  (세로)  │
│          │
│          │
└──────────┘
┌──┬──┬──┬──┐
│지│면│일│행│  ← 하단 가로 버튼
│원│접│하│정│
│  │  │기│처│
│  │  │  │리│
└──┴──┴──┴──┘
```

- 기본 활성: 알바 지원
- 버튼 클릭 시 핸드폰 화면 즉시 전환 (애니메이션 없음)

---

## 버튼 1 — 알바 지원

### 화면 A: 알바 목록
- 상단: "알바 구하기" 헤더
- 알바 카드 목록 (스크롤 가능):
  - 🍽️ 식당 서빙 — 시급 ₩9,860 / 저녁 6~10시
  - ☕ 카페 바리스타 — 시급 ₩10,500 / 오전 7~12시
  - 🏪 편의점 — 시급 ₩9,860 / 야간 11~7시
  - 🛵 배달 — 건당 ₩4,000~6,000 / 자유 시간
  - 🧹 심부름 — 건당 ₩15,000 / 자유 시간
- 카드 클릭 → 화면 B

### 화면 B: 이력서 작성 폼
- 상단: "← 뒤로" + 알바명
- 입력 항목:
  - 이름 (text input)
  - 나이 (number input)
  - 연락처 (text input)
  - 경력 (textarea, placeholder: "없으면 없음이라고 쓰세요")
  - 지원 동기 (textarea)
  - 가능 시간 (checkbox: 오전 / 오후 / 저녁 / 야간)
- 하단: "지원하기" 버튼
- 제출 시: 완료 메시지 토스트 → 목록으로 복귀

---

## 버튼 2 — 알바 면접

### 화면 A: 전화 수신 UI
- 전체 어두운 배경
- 발신자: 편의점 사장님 🧑‍💼
- 부제목: "면접 전화"
- 핸드폰 진동 애니메이션 (pulse)
- 하단: 수락(초록) / 거절(빨강) 버튼
- 거절 시: "면접 기회를 놓쳤습니다" 메시지 + 다시 걸려오기 버튼

### 화면 B: 통화 중 — 면접
- 상단 절반: 발신자 아바타 (🧑‍💼) + 이름 + "통화 중"
- 하단 절반: 음성 파형 (주파수 애니메이션, CSS 기반)
- 하단 고정: 통화 종료 버튼
- 종료 시: "면접이 끝났습니다. 결과를 기다려 주세요." 메시지

---

## 버튼 3 — 일하기

### 화면 A: 전화 수신 UI
- 발신자: 손님 (👤 번호 미표시)
- 부제목: "주문 전화"
- 수락 / 거절 버튼

### 화면 B: 통화 중 — 주문 처리
- 상단 1/3: 음성 파형 애니메이션 (텍스트 없음, 시각적으로만)
- 하단 2/3: 메뉴판
  - 메뉴 항목마다 체크박스 + 수량 조절 (- / +)
  - 항목: 아메리카노, 카페라떼, 에스프레소, 녹차라떼, 딸기스무디
  - 하단: "주문 완료" 버튼
- 주문 완료 시: 영수증 모양 요약 화면

---

## 버튼 4 — 행정 처리

### 화면 A: 행정 업무 목록
- 상단: "행정 처리" 헤더
- 업무 카드 목록:
  - 🪪 외국인 등록증 신청
  - 🏦 은행 계좌 개설
  - 📄 주민등록증 발급
  - ✈️ 여권 발급 / 갱신
  - 💸 세금 신고
  - 🏠 전입 신고
- 카드 클릭 → 화면 B

### 화면 B: 서류 양식
- 상단: "← 뒤로" + 업무명
- 업무별 입력 항목 (예: 외국인 등록증)
  - 성명 (한국어 / 영문)
  - 생년월일
  - 국적
  - 입국일
  - 체류 자격 (select: 학생 / 취업 / 방문 등)
  - 주소
- 하단: "제출하기" 버튼
- 제출 시: "접수 완료" 토스트
