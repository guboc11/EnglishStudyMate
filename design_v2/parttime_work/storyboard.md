## Scene 1 — 알바 지원

> 버튼: 알바 지원 (기본 활성)

**화면 진입 시:**
핸드폰에 알바 카드 목록이 보인다.
카드마다 업종, 시급, 근무 시간이 한국어로 적혀 있다.

**Alex가 '편의점' 카드를 탭한다.**

이력서 폼이 나타난다.

```
이름: Alex
나이: 24
연락처: 010-0000-0000
경력: [없음이라고 쓰세요]
지원 동기: [편의점 일 해보고 싶습니다...]
가능 시간: ☑ 저녁  ☑ 야간
```

**'지원하기' 버튼을 누른다.**

토스트: "지원이 완료되었습니다 ✅"
화면이 목록으로 돌아간다.

## Scene 3 — 일하기

> 버튼: 일하기

**화면 진입 시:**
어두운 화면. 핸드폰이 울린다.

```
👤
번호 미표시
주문 전화

   ○ 수락      ✕ 거절
```

**Alex가 수락 버튼을 누른다.**

화면이 전환된다. 상단에 파형, 하단에 메뉴판.

```
┌─────────────────┐
│  ≋≋≋≋≋≋≋≋≋≋≋≋  │  ← 음성 파형 (텍스트 없음)
├─────────────────┤
│  📋 메뉴판       │
│                 │
│  ☐ 아메리카노  - 1 + │
│  ☐ 카페라떼    - 0 + │
│  ☐ 에스프레소  - 0 + │
│  ☐ 녹차라떼    - 0 + │
│  ☐ 딸기스무디  - 0 + │
│                 │
│  [ 주문 완료 ]  │
└─────────────────┘
```

**Alex가 아메리카노 1개를 체크하고 주문 완료를 누른다.**

영수증 화면:
```
━━━━━━━━━━━━━━
주문 내역
━━━━━━━━━━━━━━
아메리카노 × 1   ₩4,500
━━━━━━━━━━━━━━
합계            ₩4,500
━━━━━━━━━━━━━━
감사합니다! ☕
```
