# 채팅 v2 — 플로우

## 핵심 원칙
- **AI 기능 필수** — 모든 채팅 상대는 AI 페르소나
- **진짜 핸드폰 경험** — 실제로 메시지가 "온다"는 체감이 가장 중요

---

## 진입 플로우

```
[프로토타입 채팅 진입]
    │
    ▼
[채팅 목록 화면 표시]
  - 기존 채팅방 6~7개 이미 존재 (스크롤 가능)
  - 일부는 이미 대화 기록 있음, 일부는 읽씹 상태
    │
    ▼ (1초 후)
[첫 번째 메시지 도착]
  - 빨간 뱃지 1 표시
  - 상단 푸시 배너 애니메이션 (스르르 내려왔다 올라감)
    │
    ▼ (2초 후)
[두 번째 메시지 도착]
  - 다른 사람한테서 옴
  - 뱃지 + 푸시 배너
    │
    ▼ (2.3초 후)
[세 번째 메시지 도착]
  - 또 다른 사람
  - 연달아 톡톡톡 오는 느낌
    │
    ▼
[유저 자유 탐색]
  - 채팅방 탭해서 들어가기
  - 며칠치 대화 기록 스크롤
  - 메시지 보내기
```

## 채팅방 구성 (7개)

| # | 페르소나 | 카테고리 | 대화 빈도 | 톤 |
|---|----------|----------|-----------|-----|
| 1 | 박지호 (학교 남자 친구) | 일상 | 매일 여러 번 | 반말, 활발 |
| 2 | 이수빈 (학교 여자 친구) | 일상 | 매일 | 반말, 밝고 친절 |
| 3 | 김영희 선생님 | 학교/학습 | 하루 1~2회 | 존댓말, 따뜻하지만 엄격 |
| 4 | 최사장님 (편의점 알바) | 아르바이트 | 알바 있는 날만 | 존댓말 → 짧고 직접적 |
| 5 | 구청 행정실 | 행정 | 한 달에 1~2회 | 공식적, 사무적 |
| 6 | 정민우 (동아리 친구) | 관심사 기반 | 2~3일에 한 번 | 반말, 전문 분야 대화 |
| 7 | 한소희 (알바 동료) | 아르바이트 | 알바 전후 | 반말, 도움·조언 |

## 다른 서비스와의 연동

채팅은 유저의 행동이 반영되는 "거울":

| 유저 행동 | 채팅 반영 |
|-----------|-----------|
| 강의 수강 | 학교 친구들이 같은 수업 들었으므로 → 수업 내용 대화 해금 |
| 알바 출근 | 알바 동료와 "오늘 그 손님..." 대화 해금 |
| 시험 합격 | 새로운 채팅방 해금 가능 (과외/멘토 등) |
| 행정 처리 | 구청 채팅방에서 결과 안내 |

**핵심**: 수강 전/알바 전에는 해당 주제 대화가 열리지 않음 → 세계에 참여할수록 대화가 풍부해짐

## 미래 방향 메모 (당장 구현 X)

### 과외/멘토 시스템
- 유저가 IBIK 2급 이상 취득 시 → "과외" 채팅방 해금
- 나보다 한국어를 못하는 사람(실제 뉴비 유저)과 매칭
- 내가 선생님 역할: 맥락 설명, 한국어 알려주기
- 실제 유저 간 매칭이므로 구현 범위가 다름 → 별도 설계 필요
- **가치**: "배우는 사람"에서 "가르치는 사람"으로 전환되는 경험
