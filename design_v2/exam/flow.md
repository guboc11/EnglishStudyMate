# 시험 v2 — Flow

## 레이아웃 구조

```
┌──────────┐
│          │
│          │
│  핸드폰  │
│  프레임  │
│  (세로)  │
│          │
│          │
└──────────┘
┌──┬──┬──┬──┬──┐
│그│받│표│짧│긴 │  ← 하단 5개 버튼
│림│아│현│답│답 │
└──┴──┴──┴──┴──┘
```

- 기본 활성: 유형 1 (그림 보고 뜻 맞추기)
- 버튼 클릭 시 화면 즉시 전환, 해당 유형 첫 문제로 초기화

---

## 유형 1 — 그림 보고 뜻 맞추기

### 문항 구성 (5문제, A/B 혼합)

| # | 이미지 | 형식 | 정답 |
|---|--------|------|------|
| 1 | 🐰 | 4지선다 | 토끼 |
| 2 | 🍲 (김치찌개 사진) | 4지선다 | 김치찌개 |
| 3 | 👘 (한복) | 4지선다 | 한복 |
| 4 | 🥘 (부대찌개 사진) | 직접 타이핑 | 부대찌개 |
| 5 | 🍢 (떡볶이) | 직접 타이핑 | 떡볶이 |

### 화면 구조

**4지선다 문제:**
```
┌─────────────────┐
│  문제 1 / 5     │
│  ━━━━━━━━─────  │  ← 진행 바
│                 │
│   🐰 🐰 🐰     │  ← 이미지 (이모지 또는 설명 카드)
│                 │
│  이것은 무엇인가요? │
│                 │
│  ① 고양이       │
│  ② 토끼  ← 정답│
│  ③ 강아지       │
│  ④ 쥐           │
└─────────────────┘
```

**타이핑 문제:**
```
┌─────────────────┐
│  문제 4 / 5     │
│  ━━━━━━━━━━━── │
│                 │
│   🥘🥘🥘       │
│                 │
│  이 음식의 이름은? │
│                 │
│  ┌─────────────┐│
│  │             ││  ← 텍스트 입력
│  └─────────────┘│
│  [ 확인 ]       │
└─────────────────┘
```

**정답/오답 피드백:**
- 정답: 초록 배경 + "정답이에요! ✅" + 다음 문제 버튼
- 오답: 빨간 배경 + "오답이에요 ❌ 정답: [정답]" + 다음 문제 버튼
- 마지막 문제 후: 결과 화면 (X/5 맞음)

## 유형 2 — 받아쓰기

### 문항 구성 (3문제)

| # | 형식 | 음성 내용 | 답 |
|---|------|-----------|----|
| 1 | 타이핑 | "김치찌개 2인분 주세요" | 김치찌개 2인분 주세요 |
| 2 | 타이핑 | "내일 오전 10시에 만나요" | 내일 오전 10시에 만나요 |
| 3 | 카메라 | "안녕하세요, 처음 뵙겠습니다" | 안녕하세요, 처음 뵙겠습니다 |

### 화면 구조

**형식 A — 타이핑:**
```
┌─────────────────┐
│  받아쓰기 1 / 3  │
│  ━━━────────── │
│                 │
│  ≋≋≋≋≋≋≋≋≋≋  │  ← 음성 파형 (1/3 영역)
│  🔴 00:03       │  ← 재생 중 표시
│                 │
│  들은 내용을    │
│  그대로 적어주세요│
│                 │
│  ┌─────────────┐│
│  │             ││
│  └─────────────┘│
│  [ 제출 ]       │
└─────────────────┘
```

**형식 B — 카메라 (손글씨):**
```
┌─────────────────┐
│  받아쓰기 3 / 3  │
│  ━━━━━━━━────  │
│                 │
│  ≋≋≋≋≋≋≋≋≋≋  │  ← 음성 파형
│                 │
│  ┌─────────────┐│
│  │  📷 카메라  ││  ← 후면 카메라 피드
│  │  (종이에 쓰기)││
│  └─────────────┘│
│  ⏱ 00:15        │  ← 카운트다운
│  [ 촬영 완료 ]  │
└─────────────────┘
```
- 촬영 완료 → AI OCR → 채점 결과 표시
- 프로토타입에서는 카메라 시뮬레이션 (회색 박스 + 안내 텍스트)

## 유형 3 — 일상 표현 고르기

### 문항 구성 (3문제, 4지선다)

| # | 문제 | 정답 |
|---|------|------|
| 1 | "약속 시간에 늦어서 죄송합니다"와 같은 뜻은? | 늦게 도착해서 미안해요 |
| 2 | "배가 고프다"를 정중하게 표현하면? | 배가 고픕니다 |
| 3 | 처음 만난 사람에게 하는 인사로 가장 자연스러운 것은? | 처음 뵙겠습니다 |

### 화면 구조

```
┌─────────────────┐
│  표현 1 / 3     │
│  ━━━────────── │
│                 │
│  다음과 같은 뜻의 │
│  문장을 고르세요. │
│                 │
│  "약속 시간에   │
│   늦어서 죄송합니다" │
│                 │
│  ① 제가 먼저    │
│     도착했어요  │
│  ② 늦게 도착해서│  ← 정답
│     미안해요    │
│  ③ 시간이 없어요│
│  ④ 다음에 만나요│
└─────────────────┘
```

## 유형 4 — 짧은 대답

### 문항 구성 (3문제)

| # | 나레이션 | 질문 | 형식 | 정답 |
|---|----------|------|------|------|
| 1 | 철수는 바나나를 먹었다 | 철수가 뭐 먹었어? | 텍스트 입력 | 바나나 |
| 2 | 오늘 날씨가 매우 춥다 | 오늘 날씨 어때? | 이지선다 (추워요 / 더워요) | 추워요 |
| 3 | 영희는 도서관에서 공부한다 | 영희가 어디 있어? | 텍스트 입력 | 도서관 |

### 화면 구조

```
┌─────────────────┐
│  짧은 대답 1 / 3 │
│  ━━━────────── │
│                 │
│  📢 나레이션    │
│  ┌─────────────┐│
│  │ 철수는 바나나││  ← 자막 스타일 박스
│  │ 를 먹었다.  ││
│  └─────────────┘│
│                 │
│  💬 질문        │
│  "철수가 뭐 먹었어?" │
│                 │
│  ┌─────────────┐│
│  │             ││  ← 텍스트 입력 or 이지선다
│  └─────────────┘│
│  [ 확인 ]       │
└─────────────────┘
```

## 유형 5 — 긴 대답

### 문항 구성 (3문제)

| # | 대화/발화 내용 | 형식 | 문제 |
|---|----------------|------|------|
| 1 | 두 사람이 주말 약속 잡는 대화 | 선택지 (긴 문장 4개) | 이 대화의 결론으로 알맞은 것은? |
| 2 | 선생님이 숙제 설명 | 자유 텍스트 | 숙제 내용을 요약하세요 |
| 3 | 친구가 여행 경험 이야기 | 선택지 (긴 문장 4개) | 친구가 여행에서 한 일로 맞는 것은? |

### 화면 구조

**선택지 형식:**
```
┌─────────────────┐
│  긴 대답 1 / 3  │
│  ━━━────────── │
│                 │
│  💬 대화        │
│  ┌─────────────┐│
│  │ A: 이번 주말 │
│  │ 뭐 해?      ││
│  │ B: 나 토요일 │
│  │ 바빠. 일요일 ││
│  │ 어때?       ││
│  │ A: 좋아!    ││
│  └─────────────┘│
│                 │
│  이 대화의 결론은? │
│  ① 둘은 토요일에 만난다        │
│  ② 둘은 일요일에 만나기로 했다 │← 정답
│  ③ A는 이번 주 바쁘다          │
│  ④ B는 여행을 간다             │
└─────────────────┘
```

**자유 텍스트 형식:**
```
┌─────────────────┐
│  긴 대답 2 / 3  │
│                 │
│  💬 발화        │
│  ┌─────────────┐│
│  │ 선생님: 내일 │
│  │ 까지 1~3쪽을 ││
│  │ 읽고 느낀점을││
│  │ 노트에 써오세요│
│  └─────────────┘│
│                 │
│  숙제 내용을    │
│  적어보세요.    │
│  ┌─────────────┐│
│  │             ││
│  │             ││
│  └─────────────┘│
│  [ 제출 ]       │
└─────────────────┘
```
