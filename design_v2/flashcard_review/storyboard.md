# 복습 v2 — Storyboard

유저: Alex (외국인, 한국 도착 3주차)
배경: 학교 2교시 국어(춘향전)를 막 끝냈다. 처음으로 복습 알림이 뜬다.

## Scene 1 — 알림 도착 (🔔)

Alex가 앱을 닫고 폰을 내려놓는다. 10분 뒤 락스크린에 알림이 뜬다.

```
┌────────────────────────────────┐
│  화요일, 2월 26일               │
│  오후 3:42                     │
│────────────────────────────────│
│  🌏 AI World        10분 전    │
│  '눈치' 기억나? 복습할 시간!   │
│  reading the room           │
│  #고전                  [1단계] │
│────────────────────────────────│
│  🌏 AI World        15분 전    │
│  '끝내주다' 기억나? 복습할 시간!│
│  awesome / amazing             │
│  #슬랭                  [1단계] │
│  ...                           │
└────────────────────────────────┘
```

Alex가 '눈치' 알림을 탭한다.

→ 앱 진입, 복습1 탭으로 이동. 토스트: "복습을 시작해요!"
