# 플래시카드 — UX 플로우

유저가 한국어 단어·표현을 플래시카드로 복습하는 여정.

---

## 전체 플로우

```
[앱 진입]
    │
    ▼
[플래시카드 홈]
  - 오늘 복습할 카드 수 (SM-2 알고리즘 기반)
  - 덱 목록
  - 연속 학습 스트릭
    │
    ├─ [오늘 복습 없음] ──→ "내일 다시 오세요" 표시
    │
    ▼
[복습 세션 시작]
  - 복습할 카드 총 수 표시
  - 예상 소요 시간
    │
    ▼
[카드 앞면 표시]
  - 한국어 단어/표현
  - 예문 (선택적 표시)
    │
    ▼
[유저 답변]
  - 카드 뒤집기 (탭)
    │
    ▼
[카드 뒷면 표시]
  - 발음 + 뜻 + 예문
    │
    ▼
[자가 평가]
  - 다시 | 어려움 | 보통 | 쉬움
    │
    ├─ [다시/어려움] ──→ 다음 카드로 (이 카드 곧 재등장)
    └─ [보통/쉬움]   ──→ 다음 카드로 (이 카드 나중에 재등장)
    │
    ▼
[다음 카드 or 세션 완료]
  - 모든 카드 완료 → 세션 완료 화면
    │
    ▼
[세션 완료]
  - 오늘 복습 통계
  - 스트릭 업데이트
  - 어려웠던 카드 모아보기
```

---

## 화면 상태

| 상태 | 설명 |
|------|------|
| `home` | 플래시카드 홈 (덱 목록, 오늘 복습) |
| `session-intro` | 복습 세션 시작 안내 |
| `front` | 카드 앞면 표시 중 |
| `back` | 카드 뒤집힌 상태 |
| `rating` | 자가 평가 선택 중 |
| `session-done` | 세션 완료 |

---

## 카드 추가 경로

| 경로 | 설명 |
|------|------|
| 강의 완료 후 | 에피소드 핵심 표현 → 플래시카드 추가 |
| 채팅 중 | 모르는 단어 탭 → 즉시 추가 |
| 수동 추가 | 덱 관리 화면에서 직접 입력 |

---

## 분기 조건

- **복습 일정**: SM-2 간격 반복 알고리즘 (자가 평가 점수 기반)
- **카드 재등장**: 어려움/다시 평가 → 같은 세션 내 재등장
- **덱 선택**: 전체 덱 통합 복습 or 특정 덱만 선택 복습
