# 플래시카드 — 스토리보드

화면별 레이아웃 요소·인터랙션·상태 설명.

---

## 화면 1: 플래시카드 홈

```
┌─────────────────────────────┐
│  플래시카드                 │
├─────────────────────────────┤
│                             │
│  🔥 7일 연속 학습 중!       │
│                             │
│ ┌───────────────────────┐   │
│ │  오늘 복습             │   │
│ │  📚 24장 남음          │   │
│ │       [지금 시작]      │   │
│ └───────────────────────┘   │
│                             │
│ 내 덱                       │
│ ┌───────────────────────┐   │
│ │ 직장 한국어 · 45장    │   │
│ │ 마지막 복습: 어제      │   │
│ └───────────────────────┘   │
│ ┌───────────────────────┐   │
│ │ 생활 표현 · 32장      │   │
│ │ 마지막 복습: 3일 전    │   │
│ └───────────────────────┘   │
│ ┌───────────────────────┐   │
│ │ + 새 덱 만들기        │   │
│ └───────────────────────┘   │
└─────────────────────────────┘
```

**요소:**
- 스트릭 표시 (연속 학습 일수)
- 오늘 복습 카드 수 + 시작 버튼
- 덱 목록: 이름 + 카드 수 + 마지막 복습일
- 새 덱 만들기 버튼

**인터랙션:**
- 오늘 복습 시작 → 전체 복습 필요 카드 세션
- 덱 탭 → 해당 덱만 복습 세션

## 화면 2: 카드 앞면

```
┌─────────────────────────────┐
│ ← 복습 중          12 / 24  │
│ ████████████░░░░░░░░░░      │
├─────────────────────────────┤
│                             │
│                             │
│  ┌─────────────────────┐   │
│  │                     │   │
│  │                     │   │
│  │    보고드리다        │   │
│  │                     │   │
│  │                     │   │
│  └─────────────────────┘   │
│                             │
│      탭해서 뒤집기 👆       │
│                             │
└─────────────────────────────┘
```

**요소:**
- 상단: 뒤로가기 + 진행 카운터 + 진행 바
- 카드 영역 (중앙): 한국어 단어/표현 (큰 텍스트)
- 하단: "탭해서 뒤집기" 안내

**인터랙션:**
- 카드 탭 → 뒤집기 애니메이션 → 뒷면 표시

## 화면 3: 카드 뒷면 + 자가 평가

```
┌─────────────────────────────┐
│ ← 복습 중          12 / 24  │
│ ████████████░░░░░░░░░░      │
├─────────────────────────────┤
│                             │
│  ┌─────────────────────┐   │
│  │    보고드리다        │   │
│  │  ─────────────────  │   │
│  │  [보고드리다]        │   │
│  │  격식을 갖춰 보고할  │   │
│  │  때 쓰는 표현        │   │
│  │                     │   │
│  │  예: "팀장님, 결과를 │   │
│  │  보고드립니다."      │   │
│  └─────────────────────┘   │
│                             │
│ [다시] [어려움] [보통] [쉬움] │
└─────────────────────────────┘
```

**요소:**
- 카드 뒷면: 발음(필요시) + 뜻 + 예문
- 자가 평가 버튼 4단계: 다시 / 어려움 / 보통 / 쉬움

**인터랙션:**
- 평가 버튼 탭 → 다음 카드로 이동
- 다시/어려움 선택 → 이 카드 곧 재등장

## 화면 4: 세션 완료

```
┌─────────────────────────────┐
│                             │
│        🎊                   │
│   오늘 복습 완료!            │
│                             │
│  24장 완료                  │
│  ✅ 쉬움 15장               │
│  🟡 보통 6장                │
│  🔴 어려움 3장              │
│                             │
│  🔥 8일 연속 학습!           │
│                             │
│  [어려웠던 카드 보기]        │
│  [홈으로]                   │
└─────────────────────────────┘
```

**요소:**
- 완료 축하 아이콘
- 세션 통계 (쉬움/보통/어려움 카드 수)
- 스트릭 업데이트 표시
- 어려웠던 카드 모아보기 버튼
- 홈으로 버튼
