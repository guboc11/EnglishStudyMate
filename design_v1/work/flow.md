# 작업(일) — UX 플로우

유저가 AI 세계에서 직장·행정·생활 미션을 수행하는 여정.

---

## 전체 플로우

```
[앱 진입]
    │
    ▼
[작업 목록 화면]
  - 활성 미션 목록 (기한 있음)
  - 완료된 미션 히스토리
  - 새 미션 알림 뱃지
    │
    ├─ [새 미션 발생] ──→ 시스템 알림 ("집주인에게 연락해야 합니다")
    │
    ▼
[미션 상세 화면]
  - 미션 제목 + 설명
  - 관련 AI 페르소나
  - 완료 조건 목록
  - 기한 표시
    │
    ▼
[미션 수행]
  - 채팅으로 해결 → 채팅 화면 연결
  - 서류 작성 → 폼 입력 화면
  - 장소 방문 → 위치 정보 확인
    │
    ├─ [성공] ──→ 완료 애니메이션 + 결과 텍스트
    ├─ [실패] ──→ 힌트 제공 + 재시도 유도
    └─ [기한 초과] ──→ 페널티 (관계 악화, 이벤트 트리거)
    │
    ▼
[미션 완료]
  - 완료 스탬프 + 보상 설명
  - 다음 미션 연결 (스토리 연속성)
    │
    ▼
[작업 목록으로 복귀]
```

---

## 화면 상태

| 상태 | 설명 |
|------|------|
| `list` | 미션 목록 표시 중 |
| `detail` | 미션 상세 보기 |
| `in-progress` | 미션 수행 중 |
| `success` | 미션 성공 |
| `failed` | 미션 실패 / 기한 초과 |

---

## 미션 카테고리

| 카테고리 | 예시 |
|----------|------|
| 직장 | 회의 참석, 보고서 제출, 동료 부탁 처리 |
| 행정 | 주민등록 신청, 은행 계좌 개설, 계약서 서명 |
| 생활 | 집 계약, 마트 쇼핑, 식당 주문 |
| 관계 | 이웃 인사, 친구 약속, 상사 보고 |

---

## 분기 조건

- **미션 발생 트리거**: 스토리 단계 + 관계 이벤트 + 시간 경과
- **성공 조건**: 미션마다 정의 (채팅 완료, 폼 제출 등)
- **기한 초과**: 관련 AI 페르소나 감정 변화 연동
