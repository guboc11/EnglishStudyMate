# 작업(일) — 스토리보드

화면별 레이아웃 요소·인터랙션·상태 설명.

---

## 화면 1: 작업 목록

```
┌─────────────────────────────┐
│  오늘의 할 일               │
├─────────────────────────────┤
│ 📋 긴급                     │
│ ┌───────────────────────┐   │
│ │ 🔴 월세 납부           │   │
│ │ 집주인 이철수에게 연락  │   │
│ │ D-1 · 행정             │   │
│ └───────────────────────┘   │
│                             │
│ 📋 진행 중                  │
│ ┌───────────────────────┐   │
│ │ 🟡 팀 보고서 제출      │   │
│ │ 김민준 팀장에게 제출   │   │
│ │ D-3 · 직장             │   │
│ └───────────────────────┘   │
│                             │
│ 📋 완료                     │
│ ┌───────────────────────┐   │
│ │ ✅ 주민등록 신청       │   │
│ │ 완료 · 어제            │   │
│ └───────────────────────┘   │
└─────────────────────────────┘
```

**요소:**
- 섹션 구분: 긴급 / 진행 중 / 완료
- 각 항목: 상태 아이콘 + 제목 + 관련 페르소나 + 기한 + 카테고리 태그
- 기한 색상: 🔴 D-1 이내 / 🟡 D-3 이내 / ⚪ 여유

**인터랙션:**
- 항목 탭 → 미션 상세 화면

---

## 화면 2: 미션 상세

```
┌─────────────────────────────┐
│ ← 월세 납부                 │
├─────────────────────────────┤
│                             │
│ 📍 행정 미션                │
│ 기한: 내일 오후 6시         │
│                             │
│ 집주인 이철수 씨에게        │
│ 이번 달 월세를 납부해야     │
│ 합니다. 연락을 취해 계좌    │
│ 번호를 확인하고 이체하세요. │
│                             │
├─── 완료 조건 ───────────────┤
│ ☐ 이철수에게 연락           │
│ ☐ 계좌번호 확인             │
│ ☐ 이체 완료 확인            │
│                             │
├─── 관련 페르소나 ───────────┤
│ 👤 이철수 (집주인) → 채팅   │
│                             │
│        [미션 시작하기]      │
└─────────────────────────────┘
```

**요소:**
- 상단: 뒤로가기 + 미션 제목
- 카테고리 태그 + 기한
- 미션 설명 텍스트
- 완료 조건 체크리스트
- 관련 페르소나 연결 버튼 (→ 채팅 화면)
- 미션 시작 버튼

**인터랙션:**
- 페르소나 버튼 탭 → 해당 채팅방 열기
- 완료 조건 자동 체크 (채팅 완료 감지 등)

## 화면 3: 미션 성공

```
┌─────────────────────────────┐
│                             │
│        ✅                   │
│   월세 납부 완료!            │
│                             │
│  이철수 씨가 고맙다고 했어요. │
│  관계가 좋아졌습니다.        │
│                             │
│  📈 이철수 친밀도 +10        │
│                             │
│         [확인]              │
└─────────────────────────────┘
```

**요소:**
- 완료 아이콘 + 결과 텍스트
- 관계 변화 수치 표시
- 다음 미션 힌트 (선택적)
- 확인 버튼 → 작업 목록 복귀

## 화면 4: 미션 실패 / 기한 초과

```
┌─────────────────────────────┐
│                             │
│        ❌                   │
│   기한을 넘겼습니다.         │
│                             │
│  이철수 씨가 화가 났어요.    │
│  관계가 나빠졌습니다.        │
│                             │
│  📉 이철수 친밀도 -20        │
│                             │
│  [다시 시도]    [넘어가기]   │
└─────────────────────────────┘
```

**요소:**
- 실패 아이콘 + 패널티 설명
- 관계 변화 수치 (부정적)
- 재시도 또는 넘어가기 선택
