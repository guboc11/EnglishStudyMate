# 강의 — 스토리보드

화면별 레이아웃 요소·인터랙션·상태 설명.

---

## 화면 1: 강의 목록

```
┌─────────────────────────────┐
│  강의실                     │
├─────────────────────────────┤
│ 🎯 추천 강의                │
│ ┌───────────────────────┐   │
│ │ 📚 직장 한국어 기초    │   │
│ │ 회의·보고·업무 표현    │   │
│ │ ████░░░░ 40% · 8강     │   │
│ └───────────────────────┘   │
│                             │
│ 📂 카테고리                 │
│ [생활 회화] [직장] [행정]   │
│ [문법 기초]                 │
│                             │
│ 전체 강의                   │
│ ┌───────────────────────┐   │
│ │ 편의점에서 주문하기    │   │
│ │ 생활 회화 · 3강 · 15분 │   │
│ └───────────────────────┘   │
│ ┌───────────────────────┐   │
│ │ 조사 사용법 완전 정복  │   │
│ │ 문법 기초 · 5강 · 25분 │   │
│ └───────────────────────┘   │
└─────────────────────────────┘
```

**요소:**
- 추천 강의 카드 (유저 수준·미션 기반)
- 진행률 바 + 에피소드 수
- 카테고리 필터 탭
- 전체 강의 목록

**인터랙션:**
- 카드 탭 → 강의 상세
- 카테고리 탭 → 필터링

## 화면 2: 강의 상세

```
┌─────────────────────────────┐
│ ← 직장 한국어 기초          │
├─────────────────────────────┤
│ 직장에서 자주 쓰는          │
│ 한국어 표현을 배워요.       │
│ 회의, 보고, 동료와의 대화.  │
│                             │
│ ████░░░░ 40% 완료           │
│                             │
├─── 커리큘럼 ────────────────┤
│ ✅ 1강 인사 표현 (완료)     │
│ ✅ 2강 회의 시작하기 (완료) │
│ ▶  3강 보고하기 (진행 예정) │
│ ○  4강 부탁·요청 표현       │
│ ○  5강 마무리 인사          │
│                             │
│         [이어서 학습]       │
└─────────────────────────────┘
```

**요소:**
- 강의 설명
- 전체 진행률 바
- 에피소드 커리큘럼 목록 (완료/현재/미수강 상태)
- 이어서 학습 버튼 (다음 미수강 에피소드로 이동)

## 화면 3: 에피소드 시청

```
┌─────────────────────────────┐
│ ← 3강 보고하기         🔖  │
├─────────────────────────────┤
│                             │
│  ┌─────────────────────┐   │
│  │                     │   │
│  │   [영상 플레이어]    │   │
│  │   or 텍스트 콘텐츠  │   │
│  │                     │   │
│  └─────────────────────┘   │
│  자막: "팀장님, 보고드립니다." │
│  번역: [보기/숨기기]        │
│                             │
│  💡 핵심 표현               │
│  "보고드리다" — 격식체 보고  │
│                             │
│  ─────────────────────      │
│  ◀◀  ⏸  ▶▶    00:45/03:20  │
└─────────────────────────────┘
```

**요소:**
- 영상 또는 텍스트 콘텐츠 영역
- 한국어 자막 (항상 표시)
- 번역 토글 버튼
- 핵심 표현 강조 카드
- 재생 컨트롤

**인터랙션:**
- 번역 토글 → 모국어 번역 표시/숨기기
- 핵심 표현 탭 → 상세 설명 팝업
- 퀴즈 구간 도달 → 자동으로 퀴즈 화면 전환

## 화면 4: 퀴즈

```
┌─────────────────────────────┐
│ 잠깐! 확인해볼까요?         │
├─────────────────────────────┤
│                             │
│ 팀장에게 결과를 알릴 때     │
│ 알맞은 표현은?              │
│                             │
│ ① 팀장님, 말씀드릴게요.    │
│ ② 팀장, 보고해.            │
│ ③ 팀장님, 보고드립니다.    │
│ ④ 팀장, 알려줄게요.        │
│                             │
└─────────────────────────────┘
```

**요소:**
- 퀴즈 질문
- 선택지 (4지선다)
- 선택 시: 정답/오답 피드백 + 설명
