# 강의 — UX 플로우

유저가 한국어 강의를 수강하는 여정.

---

## 전체 플로우

```
[앱 진입]
    │
    ▼
[강의 목록 화면]
  - 카테고리별 강의 목록
  - 진행률 표시
  - 추천 강의 (유저 수준 기반)
    │
    ▼
[강의 상세 화면]
  - 강의 제목 + 설명
  - 커리큘럼 목록 (에피소드)
  - 예상 소요 시간
  - 선수 강의 안내
    │
    ▼
[에피소드 진입]
  - 에피소드 제목
  - 영상 or 텍스트 콘텐츠
    │
    ▼
[학습 콘텐츠 시청]
  - 한국어 자막 + 번역 토글
  - 핵심 표현 강조 표시
  - 일시정지 / 재생
    │
    ├─ [퀴즈 등장] ──→ 인터랙티브 퀴즈 화면
    │       │
    │       ├─ [정답] ──→ 피드백 + 계속
    │       └─ [오답] ──→ 설명 + 재시도
    │
    ▼
[에피소드 완료]
  - 핵심 표현 요약
  - 플래시카드 추가 (선택)
    │
    ▼
[다음 에피소드 or 강의 완료]
  - 강의 완료 시: 수료 표시
```

---

## 화면 상태

| 상태 | 설명 |
|------|------|
| `list` | 강의 목록 표시 중 |
| `detail` | 강의 상세 보기 |
| `playing` | 에피소드 시청 중 |
| `quiz` | 퀴즈 풀기 중 |
| `completed` | 에피소드/강의 완료 |

---

## 강의 카테고리

| 카테고리 | 예시 강의 |
|----------|-----------|
| 생활 회화 | 편의점에서 주문하기, 길 묻기 |
| 직장 한국어 | 보고서 작성, 회의 표현 |
| 행정 한국어 | 주민센터 방문, 서류 작성 |
| 문법 기초 | 조사 사용법, 존댓말 체계 |

---

## 분기 조건

- **추천 강의**: 유저의 채팅 오류 패턴 + 현재 미션 카테고리 기반
- **퀴즈 트리거**: 에피소드 내 정해진 구간에서 등장
- **플래시카드 추가**: 에피소드 완료 시 핵심 표현 자동 제안
