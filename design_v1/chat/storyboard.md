# 채팅 — 스토리보드

화면별 레이아웃 요소·인터랙션·상태 설명.

---

## 화면 1: 채팅 목록

```
┌─────────────────────────────┐
│  AI 세계 채팅               │
├─────────────────────────────┤
│ ┌───────────────────────┐   │
│ │ 👤 김민준 (직장 동료)  │   │
│ │ "내일 회의 있어요"     │ 3 │
│ │ 2분 전                │   │
│ └───────────────────────┘   │
│ ┌───────────────────────┐   │
│ │ 👤 박서연 (이웃)       │   │
│ │ "안녕하세요~"          │   │
│ │ 1시간 전              │   │
│ └───────────────────────┘   │
│ ┌───────────────────────┐   │
│ │ 👤 이철수 (집주인)     │   │
│ │ "월세 납부일..."       │   │
│ │ 어제                  │   │
│ └───────────────────────┘   │
└─────────────────────────────┘
```

**요소:**
- 상단 타이틀 바 (고정)
- 각 목록 항목: 아바타(이모지) + 이름 + 관계 + 마지막 메시지 미리보기 + 시간 + 읽지 않은 수 뱃지
- 탭 이동: 채팅 / 작업 / 강의 / 플래시카드

**인터랙션:**
- 항목 탭 → 채팅방 진입
- 뱃지 있는 항목 강조 표시

## 화면 2: 채팅방

```
┌─────────────────────────────┐
│ ← 김민준                🔔  │
│    직장 동료 · 친밀도 ●●●○○  │
├─────────────────────────────┤
│                             │
│  ┌──────────────────┐       │
│  │ 내일 오전 10시에  │       │
│  │ 팀 회의 있어요.   │       │
│  │ 참석할 수 있어요? │  10:32│
│  └──────────────────┘       │
│                             │
│       ┌──────────────────┐  │
│       │ 네, 참석할 수     │  │
│       │ 있어요!          │  │
│       │             10:45│  │
│       └──────────────────┘  │
│                             │
│  ┌──────────────────┐       │
│  │ 입력 중...        │  ...  │
│  └──────────────────┘       │
│                             │
├─────────────────────────────┤
│ [메시지를 입력하세요...]  📤 │
└─────────────────────────────┘
```

**요소:**
- 상단: 뒤로가기 + 페르소나 이름 + 관계 + 친밀도 표시 + 알림 설정
- 메시지 영역: 스크롤 가능, AI 메시지(왼쪽) / 유저 메시지(오른쪽)
- 말풍선: 텍스트 + 시간
- "입력 중..." 인디케이터 (AI 응답 대기 중)
- 하단: 텍스트 입력창 + 전송 버튼 (고정)

**인터랙션:**
- 메시지 전송 → AI 응답 대기 상태
- 오류 정정: AI 메시지 안에 자연스럽게 포함 ("'저는'이라고 해요~")
- 친밀도 임계값 도달 → 상단 친밀도 바 업데이트 + 토스트 메시지

## 화면 3: 지연 응답 상태

```
┌─────────────────────────────┐
│ ← 김민준                🔔  │
│    직장 동료 · 바쁨 중 🔴   │
├─────────────────────────────┤
│                             │
│  ┌──────────────────┐       │
│  │ 잠깐요, 지금 회의 │       │
│  │ 중이에요.        │  14:20│
│  └──────────────────┘       │
│                             │
│  [응답 대기 중 — 약 2시간]  │
│                             │
└─────────────────────────────┘
```

**요소:**
- 페르소나 상태 표시: 바쁨 🔴 / 한가함 🟢
- 예상 응답 시간 표시 (선택적)

---

## 화면 4: 관계 이벤트

```
┌─────────────────────────────┐
│                             │
│   ✨ 김민준과 친해졌어요!   │
│   이제 반말로 대화해요.     │
│                             │
│         [확인]              │
└─────────────────────────────┘
```

**요소:**
- 모달 오버레이
- 이벤트 설명 텍스트
- 확인 버튼 → 채팅방 복귀

**이벤트 종류:**
- 친해짐 → 반말 전환 알림
- 서운함 → "[페르소나]가 대화방을 나갔습니다" 시스템 메시지
- AI 선발신 → 채팅 목록 뱃지 + 푸시 알림
